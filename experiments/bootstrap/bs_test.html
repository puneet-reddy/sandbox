Alert-heading

Alerts can contain headings, paragraphs, horizonal lines etc. If the text in the alert is too long, bootstrap makes it wrap automatically.

Whenever you need to, be sure to use margin utilities to keep things nice and tidy. This line for example uses the mb-0 class to zero out the bottom margin of the "<p>" tag

Dismissible alerts:

The alert-dismissible class along with the data-dismiss and role properties allows an alert to be closed.

Adding the fade and show classes makes it look a bit nicer when it vanishes.
